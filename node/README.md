# NUS Canoe

## Development Setup

### Requirements

- node.js (v16)
- yarn (v1)
- google credentials (Get this from a current developer, or go to
  Google Cloud Console and generate a new set)

### Getting started

1. Navigate to the same directory as this README.
2. Run `yarn` to install all dependencies.
3. That's it.
